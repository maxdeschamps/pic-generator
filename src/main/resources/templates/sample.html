Sample

Generate

Une image est généré aléatoirement, nous n'utilisons pas de paramètres

http://localhost:8080/generate

Tag

Noel

L'image est généré avec le tag 1, nous utilisons le paramètre t

http://localhost:8080/generate?t=1

t = Interger // tag

Développement

L'image est généré avec le tag 2, nous utilisons le paramètre t

http://localhost:8080/generate?t=2

t = Interger // tag

Crop 500x500

L'image est recadrée en 500px par 500px, nous utilisons les paramètres w, h

http://localhost:8080/generate?t=1&w=500&h=500

w = Integer // width
h = Interger // height

Resize 500

L'image est redimensionnée pour qu'un de ses côtés ne dépasse pas 500px, nous utilisons les paramètres w, h et opt

http://localhost:8080/generate?t=1&w=500&h=500&opt=2

w = Integer // width
h = Interger // height
opt = Interger (1 or 2) // 1 (default) = crop; 2 = resize

Colorize image

Jaune

L'image est colorée suivant les paramètres définis, nous utilisons les paramètres r, g et b

http://localhost:8080/generate?t=1&w=500&h=600&r=50&g=50&b=0

r = Integer (from 0 to 255) // red; default = 0
g = Interger (from 0 to 255) // green; default = 0
b = Interger (from 0 to 255) // blue; default = 0

Bleu

L'image est colorée suivant les paramètres définis, nous utilisons le paramètre b

http://localhost:8080/generate?t=1&w=500&h=600&b=200

b = Interger (from 0 to 255) // blue; default = 0

Text

Un texte est ajouté au milieu de l'image, nous utilisons le paramètre text

http://localhost:8080/generate?t=1&w=500&h=600&text=Bonjour%20!

text = String // text

Text size

Un texte est ajouté au milieu de l'image est à pour taille de police 40px, nous utilisons les paramètres text et t-size

http://localhost:8080/generate?t=1&w=500&h=600&text=Bonjour%20!&t-size=40

text = String // width
t-size = Interger // text size

Text color

Un texte violet est ajouté au milieu de l'image, nous utilisons les paramètres text et t-color

http://localhost:8080/generate?t=1&w=500&h=600&text=Violet&t-size=40&t-color=530b7d

text = String // text
t-color = String // text color (hexadecimal without #)

Multiple params

Un texte violet de 40px est ajouté au milieu de l'image du tag 1 jaune recadrée 500x500, nous utilisons les paramètres t, w, h, r, g, text, text-size et t-color

http://localhost:8080/generate?t=2&w=500&h=600&text=Violet&t-size=40&t-color=530b7d

t = Integer // tag
w = Integer // width
h = Interger // height
r = Integer (from 0 to 255) // red; default = 0
g = Interger (from 0 to 255) // green; default = 0
text = String // text
t-size = Integer // text size
t-color = String // text color (hexadecimal without #)
